## ✅  Instalação

Necessário:

- [Node.js](https://nodejs.org/en/download/) qualquer versão, de preferência a lts
- Visual Studio Code ou outro editor de sua preferência
- Git

Instalação

```
$ git clone https://github.com/Jhonmt-cpu/dt-money.git
cd dt-money
yarn ou npm install
code . 
```

Enjoy!
